# agent-skills-for-video-prompt（终极优化升级版）
高效可复用的AI Agent技能库，聚焦**短视频提示词文本创作、优化、合规过审**，支持关键词触发、单/组合技能调用，适配豆包、千问、ChatGPT、LangChain等各类AI Agent/大模型平台，全程仅输出纯文字内容，兼顾创作效率与平台合规要求，后续可拓展各类AI创作相关Agent Skills。

## 仓库简介
本仓库是一套标准化、可落地的Agent Skills体系，核心解决短视频提示词**不会写、不合规、场景适配慢、创作无标准**的问题，包含**丰富的触发关键词**、**7大核心可复用技能**、**新手/老手双模式对话引导**、**全流程合规检测逻辑**及**实际落地示例**。

所有技能均支持**独立调用**和**组合调用**，按固定流程可实现从**模板搭建→比例适配→故事创作/镜头反向创作→提示词优化→过审检测+流量适配→核对校验/快速修正→内容整合→保存复用**的一体化创作，10分钟完成全流程，且全程仅生成文字版提示词及配套内容，不生成任何图片/视频可视化内容，适配国内主流短视频平台过审规则。

### 核心优势
- 🚀 **高效复用**：模板一次搭建终身复用，替换主题/细节即可快速适配新场景
- ✅ **全程合规**：全覆盖违规/侵权/敏感词/平台误判风险，输出100%可过审提示词
- 🔧 **灵活调用**：关键词一键触发，单/组合技能自由选择，支持多Agent平台部署
- 📝 **纯文本输出**：聚焦提示词文本创作全流程，无可视化内容生成，适配各类创作场景
- 📚 **新手友好**：对话式数字选项引导，零基础可直接套用，无需专业创作/镜头知识
- 📱 **多比例适配**：支持6大主流视频比例，自动匹配平台偏好
- 🎯 **流量适配**：过审后按平台流量特征做轻优化，提升内容适配性
- 📋 **代码块输出**：所有可复用内容均代码块化+复制按钮，一键复制即用

## 核心功能
### 丰富的触发关键词
输入关键词即可一键匹配对应功能，所有关键词支持独立调用，引导选项均以**数字/字母**标注，方便快速选择。包含基础功能触发词、新增快捷缩写、风格+类型组合触发词等多种类型。

### 7大核心可复用技能
所有技能均包含**高效执行流程**+**明确使用方法**+**生成限制**，形成创作闭环，组合调用可实现从0到1完整创作。
1. **Skill 1：通用短视频提示词模板搭建**：输出9s/15s/30s/60s/120s+6大比例适配型模板，超大弹性镜头设计，标注可替换内容
2. **Skill 2：场景化故事创作**：支持单/双风格创作，按比例调整描写逻辑，输出代码块化故事
3. **Skill 3：场景化短视频提示词优化**：按视频比例优化镜头构图/运镜逻辑，调用镜头模板库，快速完成优化
4. **Skill 4：全流程核对校验**：新增视频比例-镜头构图适配性核对，支持快速修正单一偏差
5. **Skill 5：内容整合与优化**：分块代码化输出，生成含比例的唯一标识码，方便复用
6. **Skill 6：违规侵权检测与全维度过审优化**：融入视频比例+平台双重过审/流量优化
7. **Skill 7：镜头要素反向生成**：根据镜头要素+比例反向生成故事/提示词，落地双向创作

### 核心规则
1. **组合调用顺序**：模板搭建→比例适配→故事创作/镜头反向创作→提示词优化→过审检测+流量适配→核对校验/快速修正→内容整合→保存复用
2. **触发规则**：输入触发关键词立即匹配功能；无触发词时，默认输出丰富紧凑风格+中密度+通用版过审+超大弹性镜头+9:16视频比例的提示词+故事
3. **生成限制**：所有技能仅生成视频提示词相关文字内容，不生成任何图片/视频可视化内容
4. **合规规则**：全覆盖国内短视频平台审核要求，规避敏感词/侵权/特写/危险动作等所有违规风险
5. **输出格式**：所有可复用内容均代码块化+复制按钮，标注清晰可直接复制

## 快速使用
### 方式1：关键词一键触发（适合有明确需求）
直接输入触发关键词，即可一键调用对应功能，快速获取结果。
**示例**：
- 输入「提示词模板」→ 获取各时长/比例适配型通用模板
- 输入「过审检测」→ 上传已有提示词，获取合规检测结论+优化后可过审版本
- 输入「引导」→ 开启分步引导流程，逐步确认创作需求
- 输入「比例适配+1:1」→ 将已有内容适配为小红书方形比例

### 方式2：对话式分步引导（适合新手/需求不明确）
输入触发词**「引导」**，按数字选项依次确认需求，系统自动组合调用技能，全程自动化完成创作+合规检测+内容整合。
#### 新手详细模式
1. 选择创作模式（新手详细模式/老手极简模式）
2. 选择创作类型（生活日常/职场办公/兴趣爱好/氛围感/职业场景等）
3. 选择视频时长（9s/15s/30s/60s/120s/其他）
4. 选择视频比例（1:1/3:4/4:3/9:16/16:9/21:9）
5. 选择核心氛围（专业干练/暖调治愈/轻松活泼等，可多选）
6. 选择创作风格（单风格/双风格组合）
7. 选择故事细节密度（高密度/中密度/低密度）
8. 选择过审优化的适配平台（抖音/快手/视频号/小红书/通用版）
9. 确认是否需要流量适配
10. 确认是否配套文字版场景故事
11. 确认是否需要精简标注

**示例**：依次输入「1→2→1→4→1→1→2→5→2→1→2」→ 生成**职场办公类+15s+9:16+专业干练氛围+丰富紧凑风格+中密度+通用版过审+配套故事**的可过审提示词文本+检测结论+核对结论。

### 方式3：老手极简模式（适合快速创作）
直接输入创作需求，格式参考「风格/双风格+类型+时长+XX比例+氛围+密度（可选）+平台（可选）+流量适配（可选）+标注（可选）」。
**示例**：
- 单风格：丰富紧凑+职场+15s+9:16+专业干练+高密度+抖音+流量适配抖音+精简标注
- 双风格：丰富紧凑+温情细腻+亲子+30s+3:4+暖调治愈+低密度+视频号
- 反向创作：镜头反向创作+2-4个镜头+5-12s+16:9+低角度慢推+极慢切+露营+松弛慵懒

### 方式4：单/组合技能调用（适合定制化创作）
1. **单技能调用**：输入「单技能调用」+ 技能编号，即可单独使用某一技能
   **示例**：输入「单技能调用 6」→ 单独使用过审检测技能
2. **组合技能调用**：输入「组合技能调用」，查阅组合逻辑后，按需求自定义组合技能
   **示例**：模板搭建（1）→ 比例适配→ 提示词优化（3）→ 过审检测（6）→ 内容整合（5）

## 多Agent平台适配
本技能体系基于**关键词触发+固定流程+纯文字输出+对话式交互**开发，适配所有支持该特性的AI Agent/大模型平台，零开发/轻开发即可落地，部分主流平台适配方式如下：

### 国内商用智能助手（零基础部署）
- **字节跳动豆包**：通过「自定义指令/指令库」导入触发关键词+技能规则，直接调用
- **阿里通义千问**：通过「自定义技能/场景对话」录入技能逻辑，设关键词为功能入口
- **百度文心一言**：通过「定制化问答/技能广场」部署，7大技能设为独立子功能
- **科大讯飞星火**：将技能制作为「指令模板」，输入关键词一键调用

### 海外商用/开源Agent框架（轻量开发）
- **OpenAI（ChatGPT/GPTs）**：创建专属GPT，将关键词设为触发指令，录入技能流程/检测规则
- **LangChain/LLaMA Index**：将7大技能封装为Tools/Functions，关键词设为工具触发词
- **Claude/AutoGPT**：录入技能全流程规则，实现对话引导+自动化技能调用
- **Gemini Advanced**：通过「自定义指令」录入关键词+技能逻辑，强化文本细节优化

### 企业级Agent平台（私有化部署）
- **字节跳动火山方舟**：将技能封装为「方舟技能」，实现企业级私有化部署
- **百度文心千帆/阿里通义灵码**：封装为API，适配团队批量提示词生产
- **智谱AI ChatGLM Agent**：深度适配中文场景，部署为贴合国内平台的专属创作Agent

## 高效复用技巧
1. **场景快速适配**：已有成品提示词，仅替换**主题+专属细节**，调用「过审检测」即可适配新场景，10分钟完成
2. **模板复用**：通用模板一次搭建，通过「提示词优化」快速适配不同场景/氛围/时长/比例
3. **比例一键切换**：输入「比例适配+XX比例+标识码」，10秒完成全流程比例适配
4. **过审检测单独用**：无论是否为本技能创作的提示词，均可调用「过审检测」做合规优化，精准规避平台审核风险
5. **团队标准化**：将技能流程/模板复制到团队操作手册，统一创作标准，降低新人学习成本
6. **批量生产**：按组合调用流程，批量生成不同主题/时长/氛围/比例的提示词，适配多账号/多品牌运营需求
7. **双向创作**：支持「故事→镜头→比例」正向创作和「镜头要素+比例→故事/提示词」反向创作
