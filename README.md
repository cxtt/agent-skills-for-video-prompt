# agent-skills-for-video-prompt
高效可复用的AI Agent技能库，聚焦**短视频提示词文本创作、优化、合规过审**，支持关键词触发、单/组合技能调用，适配豆包、千问、ChatGPT、LangChain等各类AI Agent/大模型平台，全程仅输出纯文字内容，兼顾创作效率与平台合规要求，后续可拓展各类AI创作相关Agent Skills。

## 仓库简介
本仓库是一套标准化、可落地的Agent Skills体系，核心解决短视频提示词**不会写、不合规、场景适配慢、创作无标准**的问题，包含**12个核心触发关键词**、**6大核心可复用技能**、**对话式分步引导流程**、**全流程合规检测逻辑**及**实际落地示例**。

所有技能均支持**独立调用**和**组合调用**，按固定流程可实现从**模板搭建→故事创作→提示词优化→合规检测→核对校验→内容整合**的一体化创作，10分钟完成全流程，且全程仅生成文字版提示词及配套内容，不生成任何图片/视频可视化内容，适配国内主流短视频平台过审规则。

### 核心优势
- 🚀 **高效复用**：模板一次搭建终身复用，替换主题/细节即可快速适配新场景
- ✅ **全程合规**：全覆盖违规/侵权/敏感词/平台误判风险，输出100%可过审提示词
- 🔧 **灵活调用**：关键词一键触发，单/组合技能自由选择，支持多Agent平台部署
- 📝 **纯文本输出**：聚焦提示词文本创作全流程，无可视化内容生成，适配各类创作场景
- 📚 **新手友好**：对话式数字选项引导，零基础可直接套用，无需专业创作/镜头知识

## 核心功能
### 12个核心触发关键词
输入关键词即可一键匹配对应功能，所有关键词支持独立调用，引导选项均以**数字/字母**标注，方便快速选择。

| 触发关键词 | 核心作用 | 适用场景 |
| ---- | ---- | ---- |
| 技能使用说明&示例 | 输出技能完整使用规则、调用方式及多场景实操示例 | 初次使用、查阅操作指南/实操案例 |
| 引导 | 触发短视频提示词创作分步引导流程，确认创作类型/时长/氛围等 | 需求不明确、新手创作，需逐步确认要素 |
| 生成冲浪视频的提示词 | 生成合规过审的冲浪主题短视频提示词，同步完成违规检测 | 快速获取冲浪主题可过审提示词 |
| 帮我生成5个连贯场景的保险销售人员的工作 | 创作保险销售5个连贯场景故事+对应提示词，全程合规 | 职业场景故事+提示词一体化创作 |
| 单技能调用 | 输出各独立技能使用方法，支持指定技能编号调用 | 仅需使用单个核心技能，精准调用 |
| 组合技能调用 | 展示技能组合调用逻辑+实操流程，支持自定义组合 | 按需组合多技能，实现全流程创作 |
| 过审检测 | 对已有提示词做全维度合规检测，输出优化后可过审版本 | 已有提示词，需合规检测/过审优化 |
| 提示词模板 | 输出15s/30s/60s通用短视频提示词模板，标注可替换内容 | 快速搭建通用框架，适配多场景 |
| 场景故事创作 | 根据指定主题创作贴合调性的文字版配套故事，与提示词呼应 | 为提示词创作配套故事，强化场景贴合度 |
| 提示词优化 | 对通用提示词按场景/氛围做细节适配，提升画面感 | 已有通用模板，需适配具体场景/氛围 |
| 内容整合 | 将故事/提示词/检测结论/核对清单整合为统一格式文档 | 创作完成后，统一格式存档/复用/对接 |
| 核对校验 | 对提示词做全流程核对，修正偏差并输出核对结论 | 生成提示词后，校验完整性/适配性/合规性 |

### 6大核心可复用技能
所有技能均包含**高效执行流程**+**明确使用方法**+**生成限制**，形成创作闭环，组合调用可实现从0到1完整创作。
1. **Skill 1：通用短视频提示词模板搭建**：输出15/30/60s通用模板，固定镜头时长+核心要素，标注可替换内容
2. **Skill 2：场景化故事创作**：1分钟定核心，拆3-5个连贯场景，创作与提示词高度呼应的文字版故事
3. **Skill 3：场景化短视频提示词优化**：30秒分析场景，快速适配镜头/拍摄/音乐/细节，1分钟完成优化
4. **Skill 4：全流程核对校验**：按预设标准逐点核对，修正偏差，输出文字版核对结论+修正后提示词
5. **Skill 5：内容整合与优化**：按固定结构整合所有创作内容，删冗余/统格式/补衔接，生成可复用文档
6. **Skill 6：违规侵权检测与全维度过审优化**：全覆盖敏感词/违规动作/版权/平台误判风险，自动去敏感化，输出100%可过审提示词

### 核心规则
1. **组合调用顺序**：模板搭建→故事创作→提示词优化→违规侵权检测与过审优化→核对校验→内容整合
2. **触发规则**：输入触发关键词立即匹配功能；无触发词时，默认输出合规可过审的视频提示词文本
3. **生成限制**：所有技能仅生成视频提示词相关文字内容，不生成任何图片/视频可视化内容
4. **合规规则**：全覆盖国内短视频平台审核要求，规避敏感词/侵权/特写/危险动作等所有违规风险

## 快速使用
### 方式1：关键词一键触发（适合有明确需求）
直接输入上述**12个核心触发关键词**，即可一键调用对应功能，快速获取结果。
**示例**：
- 输入「提示词模板」→ 获取15/30/60s通用短视频提示词模板
- 输入「过审检测」→ 上传已有提示词，获取合规检测结论+优化后可过审版本
- 输入「引导」→ 开启分步引导流程，逐步确认创作需求

### 方式2：对话式分步引导（适合新手/需求不明确）
输入触发词**「引导」**，按数字选项依次确认需求，系统自动组合调用技能，全程自动化完成创作+合规检测+内容整合。
#### 引导流程
1. 选择创作类型（生活日常/职场办公/兴趣爱好/氛围感/职业场景/其他）
2. 选择视频时长（15s/30s/60s/其他）
3. 选择核心氛围（专业干练/暖调治愈/轻松活泼等，可多选）
4. 确认是否配套文字版场景故事
5. 系统自动执行全流程，输出整合后的可过审提示词文档

**示例**：依次输入「2→1→1→1」→ 生成**职场办公类+15s+专业干练氛围+配套故事**的可过审提示词文本+检测结论+核对结论。

### 方式3：单/组合技能调用（适合定制化创作）
1. **单技能调用**：输入「单技能调用」+ 技能编号，即可单独使用某一技能
   **示例**：输入「单技能调用 6」→ 单独使用过审检测技能
2. **组合技能调用**：输入「组合技能调用」，查阅组合逻辑后，按需求自定义组合技能
   **示例**：模板搭建（1）→ 提示词优化（3）→ 过审检测（6）→ 内容整合（5）

## 多Agent平台适配
本技能体系基于**关键词触发+固定流程+纯文字输出+对话式交互**开发，适配所有支持该特性的AI Agent/大模型平台，零开发/轻开发即可落地，部分主流平台适配方式如下：

### 国内商用智能助手（零基础部署）
- **字节跳动豆包**：通过「自定义指令/指令库」导入触发关键词+技能规则，直接调用
- **阿里通义千问**：通过「自定义技能/场景对话」录入技能逻辑，设关键词为功能入口
- **百度文心一言**：通过「定制化问答/技能广场」部署，6大技能设为独立子功能
- **科大讯飞星火**：将技能制作为「指令模板」，输入关键词一键调用

### 海外商用/开源Agent框架（轻量开发）
- **OpenAI（ChatGPT/GPTs）**：创建专属GPT，将关键词设为触发指令，录入技能流程/检测规则
- **LangChain/LLaMA Index**：将6大技能封装为Tools/Functions，关键词设为工具触发词
- **Claude/AutoGPT**：录入技能全流程规则，实现对话引导+自动化技能调用
- **Gemini Advanced**：通过「自定义指令」录入关键词+技能逻辑，强化文本细节优化

### 企业级Agent平台（私有化部署）
- **字节跳动火山方舟**：将技能封装为「方舟技能」，实现企业级私有化部署
- **百度文心千帆/阿里通义灵码**：封装为API，适配团队批量提示词生产
- **智谱AI ChatGLM Agent**：深度适配中文场景，部署为贴合国内平台的专属创作Agent

## 高效复用技巧
1. **场景快速适配**：已有成品提示词，仅替换**主题+专属细节**，调用「过审检测」即可适配新场景，10分钟完成
2. **模板复用**：通用模板一次搭建，通过「提示词优化」快速适配不同场景/氛围/时长
3. **过审检测单独用**：无论是否为本技能创作的提示词，均可调用「过审检测」做合规优化，精准规避平台审核风险
4. **团队标准化**：将技能流程/模板复制到团队操作手册，统一创作标准，降低新人学习成本
5. **批量生产**：按组合调用流程，批量生成不同主题/时长/氛围的提示词，适配多账号/多品牌运营需求
