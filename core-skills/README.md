# 核心技能库说明
本目录为**短视频提示词创作Agent Skills**核心内容，包含12个触发关键词、6大核心技能、对话引导流程、合规检测规则及落地示例，所有技能均遵循**纯文字输出、合规可过审、高效复用**原则。

## 核心能力
- 支持关键词一键触发，单/组合技能自由调用
- 全覆盖短视频提示词创作→优化→检测→整合全流程
- 适配国内主流短视频平台过审规则，输出100%可过审内容
- 全程仅生成文字版提示词及配套内容，无图片/视频可视化生成

## 目录说明
- video-prompt-skill/：短视频提示词技能全量内容，含规则、流程、示例
- 后续新增核心技能将在本目录下新建子文件夹，保持模块化

## 快速使用
1. 先查阅`video-prompt-skill/trigger-words.md`确认触发关键词
2. 新手推荐先看`video-prompt-skill/guide-steps.md`对话引导流程
3. 实际创作参考`video-prompt-skill/demo-example.md`落地示例
4. 通用模板直接复用`templates/`目录下的文件
