# 对话式引导分步流程
核心：以数字选项对话形式引导确认需求，自动组合调用对应技能，同步完成违规侵权检测与过审优化，**全程仅生成文字版短视频提示词及配套内容**，不涉及图片/视频生成，新手可直接套用。

## 核心规则
1. 引导者仅做提示词文本相关的需求确认、技能调用、结果输出；
2. 所有选项均为**数字序号**，用户直接输入数字即可，无需额外描述；
3. 支持「仅生成提示词」和「提示词+配套故事」两种创作模式；
4. 全程自动执行过审检测+核对校验，最终输出整合后的可过审文档。

## 完整对话引导流程（含交互示例）
### 角色定义
- 引导者：Agent技能调用引导方（如豆包/千问/ChatGPT）
- 使用人：需求发起方（创作者/团队）

### 流程步骤
1. **创作类型选择**
引导者：您好！我将引导您生成**短视频提示词文本**，并自动做违规侵权 + 过审检测，全程仅输出文字版提示词及相关配套内容，无图片、视频生成。请选择你要创作的视频提示词类型（直接输入数字即可）：
1. 生活日常类（如居家、出行、探店）
2. 职场办公类（如工作日常、技能提升、职场复盘）
3. 兴趣爱好类（如美食、手工、运动、摄影、冲浪）
4. 氛围感类（如治愈、松弛、氛围感vlog）
5. 职业场景类（如保险销售、教师、医护、外卖员）
6. 其他类型（请后续补充具体描述）
使用人：【输入数字，示例：2】

2. **视频时长选择**
引导者：请选择你需要的视频提示词时长（直接输入数字即可）：
1. 15s（默认推荐，适配短视频主流时长）
2. 30s
3. 60s
4. 其他时长（请后续补充具体数字）
使用人：【输入数字，示例：1】

3. **核心氛围选择**
引导者：请选择视频提示词的核心氛围（直接输入数字即可，可多选）：
1. 专业干练 2. 暖调治愈 3. 轻松活泼 4. 静谧沉淀 5. 简约高级 6. 温馨日常 7. 松弛慵懒 8. 热血动感 9. 其他氛围（请后续补充具体描述）
使用人：【输入数字，示例：1】

4. **配套故事确认**
引导者：是否需要为视频提示词配套创作文字版场景故事（直接输入数字即可）：
1. 需要（配套故事让提示词更贴合场景，无需额外补充细节）
2. 不需要，仅生成提示词文本
使用人：【输入数字，示例：1/2】

5. **自动执行全流程+结果输出**
#### 场景1：使用人选择「需要配套故事」（示例输入：2→1→1→1）
引导者：需求已确认（职场办公类、15s、专业干练氛围、需配套文字版故事），开始执行全流程，**全程仅输出文字版内容**：
1. 调用 Skill 1 搭建15s通用短视频提示词模板文本
2. 调用 Skill 2 创作配套文字版场景故事
3. 调用 Skill 3 适配场景优化生成视频提示词文本
4. 调用 Skill 6 违规侵权检测与过审优化，检测敏感词、色情、暴露、低俗、违规动作、儿童、特写、版权、侵权、易误判内容并自动去敏感化优化，输出文字版检测结论+优化后提示词文本
5. 调用 Skill 4 全流程核对校验，输出文字版核对结论+修正后提示词文本
6. 调用 Skill 5 整合输出最终版可过审提示词文本文档，请稍等～
【执行完成后】
引导者：全部操作完成！已为您生成职场办公（15s、专业干练、可过审）的**短视频提示词文本及配套文字版故事**，整合后的文字版文档已同步，您可直接使用～

#### 场景2：使用人选择「不需要配套故事」（示例输入：2→1→1→2）
引导者：收到！需求确认（职场办公类、15s、专业干练氛围、仅生成提示词文本），开始执行全流程，**全程仅输出文字版内容**：

3. 调用 Skill 6 违规侵权检测与过审优化，输出文字版检测结论+优化后提示词文本

【执行完成后】
引导者：操作完成！已为您生成职场办公（15s、专业干练、可过审）的短视频提示词文本，整合后的文字版文档已同步，您可直接使用～
