# Skill 6 过审检测&合规优化核心规则
本规则为短视频提示词创作的**合规核心标准**，适配国内抖音、快手、小红书、视频号等主流短视频平台审核要求，覆盖所有违规/侵权/误判风险点，是生成可过审提示词的关键依据。

## 一、全维度检测范围（无遗漏）
### 1. 内容敏感类
- 敏感词、违规词、网络黑话、不良表述
- 低俗、色情、暴露、衣着暴露相关描述
- 违背公序良俗、价值观的内容表述

### 2. 行为违规类
- 儿童、未成年人相关的拍摄/动作描述
- 危险动作、危险行为、教学类危险操作
- 平台禁止的各类违规动作、姿势描述

### 3. 画面特写类
- 身体局部特写（手部、面部、肢体等过度特写）
- 易引发平台误判的特写镜头描述
- 特写镜头占比过高、无场景铺垫的特写描述

### 4. 版权侵权类
- 品牌LOGO、商标、知名IP相关描述
- 影视截图、版权素材、付费内容相关描述
- 侵权画面、盗用素材相关描述

### 5. 平台误判类
- 易触发关键词审核的表述
- 易被系统误判为违规的姿势/动作描述
- 与版权库高度相似的画面描述

## 二、自动优化标准（100%可过审）
### 1. 敏感内容优化
- 直接删除所有低俗、暴露、违规的内容描述
- 用**中性、合规表述**替换敏感关键词，不改变核心场景
- 规避所有网络黑话、不良表述，使用通用书面语

### 2. 违规行为优化
- 删除所有儿童、未成年人相关的拍摄/动作描述
- 替换危险动作描述为**安全、合规的同类动作**
- 移除所有教学类危险操作的表述

### 3. 特写镜头优化
- 避免过度特写，**增加场景铺垫**，特写与全景/中景结合
- 简化手部、面部等局部特写的描述，降低误判概率
- 减少特写镜头占比，优先使用中景、全景镜头

### 4. 版权侵权优化
- 完全删除所有品牌LOGO、商标、知名IP的描述
- 移除影视截图、版权素材相关表述，改用原创场景描述
- 避免使用与版权库高度相似的画面描述，增加个性化细节

### 5. 系统误判优化
- 拆解易触发审核的长关键词，改用短句描述
- 调整易误判的姿势/动作表述，更贴合日常场景
- 优化画面描述的用词，避免与平台违规库匹配

## 三、检测结论输出标准
检测结论为**文字版固定格式**，明确标注各风险点检测结果，最终结论仅分两种：
1. **可直接使用**：无任何违规/侵权/误判风险，提示词文本无需优化；
2. **优化后使用**：标注具体风险点，输出优化后的可过审提示词文本。

## 四、系统误伤处理方法
若内容本身合规，但出现**无法生成/审核不通过/画面屏蔽**，大概率为系统误伤，处理方法：
1. 记录具体的**违规提示信息**（如关键词触发、特写误拦）；
2. 针对误伤点做**针对性表述优化**（如拆解关键词、简化特写描述）；
3. 重新生成提示词文本，再次执行过审检测；
4. 所有误伤优化均不改变核心场景，仅调整表述方式。
