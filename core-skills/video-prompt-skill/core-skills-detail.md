# 7大核心可复用技能详情（终极优化升级版）
所有技能均包含**高效执行流程**+**明确使用方法**+**严格生成限制**，全程仅输出**文字版内容**，无任何图片/视频可视化生成操作，支持独立/组合调用。

## Skill 1：通用短视频提示词模板搭建（风格+时长+比例适配+超大弹性镜头全版）
### 高效流程
固定全时长框架→**选定视频比例+匹配构图规则**→明确风格适配规则→设定超大弹性镜头区间→动态预留镜头位→整合风格化特技→简化替换→补充适配提示
### 生成限制
仅输出9s/15s/30s/60s/120s+**1:1/3:4/4:3/9:16/16:9/21:9**风格比例适配型短视频提示词模板文本，无图片/视频生成操作
### 使用方法
搭建模板时，先固定对应时长+**指定视频比例（默认9:16）**基础框架，明确7大风格/双风格融合对应的**超大弹性镜头数量范围、弹性时长区间、基础切换节奏+比例专属构图规则**，预留可增删的镜头位，按风格+比例整合常用特技并补充提示，用「{}」标注可替换内容+镜头时长调整位，一次搭建可动态复用。
### 核心升级
新增6大视频比例构图框架，匹配不同平台/创作场景的比例需求；9:16为默认比例，强化竖版短视频主流创作逻辑。

## Skill 2：场景化故事创作（多风格+双融合+比例适配+情节编排+细节密度调节版）
### 高效流程
定核心→选风格/双风格→**选定视频比例**→选细节密度→拆场景→标注情节编排→强呼应→统调性→速收尾→**代码块化输出+复制按钮**
### 生成限制
仅输出与视频提示词配套的**风格/双风格+密度+比例+编排标注**文字版场景故事，**以代码块格式输出+复制按钮**，无图片/视频生成操作
### 使用方法
1分钟确定故事核心+创作风格/双风格+**视频比例（默认9:16）**+细节密度，拆分为3-5个连贯场景，**按比例调整描写逻辑（宽屏重场景，竖版重主体）**，标注每个场景的重点情节/过渡情节/核心画面，故事描述紧扣提示词细节，用「总起+场景衔接+总结」完成创作，最后以**可直接复制的代码块**输出，为超大弹性镜头设计提供精准依据。
### 7大故事创作风格标准/3档细节密度调节
保留原有所有规则，**同步适配6大视频比例**，不同比例对应不同的描写侧重点。
### 核心升级
支持视频比例适配创作，输出格式改为代码块化+复制按钮，标记清晰可直接复制内容，提升使用效率。

## Skill 3：场景化短视频提示词优化（风格+双融合+比例适配+超大弹性镜头+模板库版）
### 高效流程
快分析创作要素（场景/氛围/风格/密度/编排）→**匹配视频比例构图规则**→选模板/套框架→设计超大弹性镜头/调用镜头模板→秒适配音画运镜→强衔接→**代码块化输出+复制按钮**
### 生成限制
仅输出优化后的**风格/双风格+编排+比例+超大弹性镜头**场景化视频提示词文本，**以代码块格式输出+复制按钮**，无图片/视频生成操作
### 使用方法
30秒分析创作核心要素，**明确视频比例（默认9:16）**，可直接调用「镜头模板库+XX场景」的现成设计（自带比例构图规则）或套用模板基础框架，按风格/双风格融合规则+**比例构图规则**确定超大弹性镜头数量范围+弹性时长区间，再按情节编排为重点情节/核心画面分配长时长、过渡情节分配短时长，匹配对应镜头特技/拍摄效果/专属运镜/音乐，补充场景细节和衔接语句，1分钟完成优化，最后以**可直接复制的代码块**输出。
### 7大风格超大弹性镜头设计规则
保留原有所有规则，**新增6大比例构图/运镜适配细则**，不同风格+不同比例匹配专属的镜头设计逻辑。
### 双风格融合镜头设计核心规则
保留原有所有规则，**新增比例融合适配规则**，主风格定调性，副风格补细节，比例定构图核心。
### 核心升级
按视频比例优化镜头设计逻辑，输出格式改为代码块化+复制按钮，实现一键复制即用。

### 生成限制
仅输出文字版核对校验结论及修正后的提示词文本，**修正后提示词为代码块格式+复制按钮**，无图片/视频生成操作
### 使用方法
按「**模板适配→视频比例-镜头构图/运镜适配→风格/双融合-超大镜头适配→编排-时长适配→密度-画面适配→音画适配→卡点要素→衔接流畅→合规性适配**」顺序逐点核对，发现偏差快速修正，标注「适配点+修正点」；若仅单一偏差，可直接调用「快速修正+问题点」（支持比例相关偏差），无需全流程核对，确保提示词与故事风格/密度/编排/**比例**高度契合，修正后的最终提示词以**代码块格式+复制按钮**输出。
### 核心核对维度
原有基础上**新增「视频比例-镜头构图适配性」「视频比例-运镜适配性」两大核心维度**，实现全维度无死角核对。
### 核心升级
新增视频比例相关核对维度，支持比例偏差快速修正，修正后内容代码块化+复制按钮。

### 高效流程

### 生成限制
仅输出**比例适配+结构化代码块标注**的文字版整合文档（含风格/双风格故事、超大弹性镜头提示词、检测结论、核对清单、标识码），**所有可复用内容均为代码块格式+复制按钮**，无图片/视频生成操作
### 使用方法
按「**风格化故事模块（风格/双风格+密度+比例+编排标注）→超大弹性镜头提示词模块（设计规则+比例构图+运镜标注+音乐参考）→合规模块（过审检测+流量适配+核对结论）→复用提示模块（标识码+比例适配技巧+复用技巧）**」四大板块梳理结构，删除冗余表述，统一标题格式，补充模块衔接语句，在关键位置标注核心信息+**比例构图要点**，生成含比例的唯一标识码，最后补充同类型场景+比例的复用技巧，1分钟完成整理，**四大板块分独立代码块输出+全文档整合代码块输出，均带复制按钮**，方便快速查阅复用。
### 标注密度开关

### 核心升级

### 生成限制

### 使用方法
检测范围全覆盖敏感词/违规动作/版权风险/平台易误判内容，**同时结合视频比例审核规则**，自动判断是否触发违规/屏蔽/审核不通过，执行**保特征+比例+平台化**去敏感化优化，可选择叠加「流量适配+XX平台」（融合平台比例偏好），按平台流量特征+比例规则做轻优化，输出100%可过审且贴合原创作风格+比例的提示词文本，**以可直接复制的代码块格式输出**。

### 核心升级

## Skill 7：镜头要素反向生成（故事/提示词）【新增核心技能·比例适配升级】
### 高效流程
输入镜头要素（含比例）→自动匹配风格/密度/比例→生成故事/提示词→过审检测→**比例-构图核对校验**→整合输出→**代码块化+复制按钮**
### 生成限制
仅输出与镜头要素匹配的**文字版故事/提示词+检测结论+比例适配信息**，**故事/提示词为代码块格式+复制按钮**，无图片/视频生成操作
### 使用方法
用户仅需输入**镜头核心要素（数量+时长+运镜+切换节奏+视频比例）**，如「2-4个镜头+5-12s时长+低角度慢推+极慢切+9:16+露营+松弛慵懒」，无比例时默认9:16，系统自动匹配对应风格+适配密度+**比例构图规则**，生成贴合的故事/提示词，全程遵循原有超大弹性镜头规则，支持过审检测+流量适配+**比例-构图核对校验**全流程联动，最终生成的故事/提示词以**代码块格式+复制按钮**输出。
### 核心匹配逻辑
镜头要素+**视频比例**→风格匹配→密度适配→故事/提示词生成，确保镜头+内容+**比例**高度契合，落地双向创作需求。
### 核心升级
支持输入镜头要素时指定视频比例，全流程适配比例规则，输出内容代码块化+复制按钮。
