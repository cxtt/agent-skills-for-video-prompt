# 6大核心可复用技能详情
所有技能均包含**高效执行流程**+**明确使用方法**+**严格生成限制**，全程仅输出**文字版内容**，无任何图片/视频可视化生成操作，支持独立/组合调用。

## Skill 1：通用短视频提示词模板搭建
### 高效流程
固定框架 → 明确要素 → 整合特技 → 简化替换 → 补充提示
### 生成限制
仅输出15s/30s/60s短视频提示词模板文本，无图片/视频生成操作
### 使用方法
1. 固定4个镜头的时长分配（适配对应总时长）；
2. 明确每个镜头的5大核心要素（镜头角度/动作、拍摄效果、色调、音乐、氛围）；
3. 整合短视频常用镜头特技、切换方式，补充场景适配提示；
4. 用「[ ]」标注所有可替换内容，模板一次搭建，终身复用。

## Skill 2：场景化故事创作
### 高效流程
定核心 → 拆场景 → 强呼应 → 统调性 → 速收尾
### 生成限制
仅输出与视频提示词配套的文字版场景故事，无图片/视频生成操作
### 使用方法
1. 1分钟确定故事核心（如“职场成长”“兴趣体验”“职业日常”）和整体脉络；
2. 拆分为3-5个连贯场景，**每个场景对应1个短视频提示词**；
3. 故事描述紧扣提示词细节，与镜头、氛围高度呼应；
4. 统一全故事调性（如暖调/专业/治愈/轻松）；
5. 用「总起+总结」快速完成开头结尾，无需复杂构思，简洁贴合场景即可。

## Skill 3：场景化短视频提示词优化
### 高效流程
快分析 → 套模板 → 秒适配（镜头/拍摄/音乐/细节）→ 强衔接
### 生成限制
仅输出优化后的场景化视频提示词文本，无图片/视频生成操作
### 使用方法
1. 30秒分析目标场景的**核心氛围**（如静谧/热闹/专业干练）和**创作重点**；
2. 直接套用Skill1搭建的通用提示词模板，不修改框架；
3. 快速匹配对应镜头特技、拍摄效果、音乐风格，补充**场景专属细节**；
4. 增加镜头间、场景间的衔接语句，提升流畅度；
5. 1分钟完成优化，无需重新创作。

## Skill 4：全流程核对校验
### 高效流程
用模板 → 逐点核 → 速修正 → 出结论
### 生成限制
仅输出文字版核对校验结论及修正后的提示词文本，无图片/视频生成操作
### 使用方法
1. 复用预设的核对标准，以通用模板为基准；
2. 按**模板适配→卡点要素→衔接流畅→细节适配→合规性适配**顺序逐点核对；
3. 发现偏差（如模板偏离、卡点不准、衔接生硬、合规细节遗漏）快速修正；
4. 整理完整的文字版核对结论，明确标注「是否可直接使用」；
5. 输出修正后的最终提示词文本。

## Skill 5：内容整合与优化
### 高效流程
理结构 → 删冗余 → 统格式 → 补衔接 → 保核心
### 生成限制
仅输出文字版整合文档（含故事、提示词、检测结论、核对清单），无图片/视频生成操作
### 使用方法
1. 按**故事→提示词→过审检测结论→核对清单**的固定顺序梳理结构；
2. 删除多余空锚点、重复表述、无效细节，精简内容；
3. 统一全文档的标题、格式、标注方式，保持视觉整洁；
4. 补充模块间的衔接语句，让文档逻辑更连贯；
5. 完整保留所有核心内容，1分钟完成整理，方便查阅、复用、跨团队对接。

## Skill 6：违规侵权检测与全维度过审优化（完整版）
### 高效流程
查关键词 → 扫风险 → 去敏感化 → 改表述 → 避特写 → 换描述 → 重生成
### 生成限制
仅输出文字版检测结论及优化后的可过审视频提示词文本，无图片/视频生成操作
### 使用方法
#### 第一步：全范围风险检测
覆盖以下所有风险点，无遗漏：
1. 敏感词、违规词、低俗、色情、暴露、衣着暴露；
2. 儿童、未成年人、危险动作、教学动作、局部特写；
3. 版权素材、侵权画面、品牌LOGO、影视截图；
4. 平台易误判内容：手部特写、身体局部、姿势动作。

#### 第二步：风险判定
自动判断内容是否会触发：
违规、侵权、无法生成、画面屏蔽、审核不通过。

#### 第三步：自动去敏感化优化
1. 删除暴露、低俗、危险、特写等违规内容描述；
2. 替换所有敏感关键词，改用合规表述；
3. 调整镜头角度、人物姿势、画面描述，规避平台误判；
4. 移除所有版权相关、侵权风险的描述；
5. 对系统易误伤的内容做针对性优化，避免关键词触发审核。

#### 第四步：输出结果
1. 详细的文字版检测结论（标注各风险点检测结果）；
2. 100%可过审、不触发平台限制的最终提示词文本。

### 补充说明
内容本身正常、合规、安全，出现无法生成一般是**系统误伤**：关键词触发审核、特写误拦、版权库相似误判。可告知具体违规提示的页面/功能，直接优化为能过审的提示词文本版本。
