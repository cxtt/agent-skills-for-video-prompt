# 12个核心触发关键词说明
所有关键词支持**独立调用**，也可按固定顺序**组合调用**，输入关键词立即匹配功能，引导选项均以**数字/字母**标注，方便快速选择。
| 触发关键词 | 核心作用 | 适用场景 |
| ---- | ---- | ---- |
| 技能使用说明&示例 | 输出本套技能的完整使用规则、调用方式及多场景实操示例，清晰展示单技能/组合技能落地用法 | 初次使用、需查阅完整操作指南、参考实操案例时 |
| 引导 | 触发短视频提示词创作的分步引导流程，通过数字选项目标化确认创作类型、时长、氛围等核心需求 | 需求不明确、新手创作，需逐步确认创作核心要素时 |
| 生成冲浪视频的提示词 | 直接生成合规可过审的冲浪主题短视频提示词文本，同步完成违规侵权检测与过审优化 | 需快速获取冲浪主题、合规可过审的短视频提示词时 |
| 帮我生成5个连贯场景的保险销售人员的工作 | 创作保险销售工作相关的5个连贯场景文字版故事，同时生成对应场景的短视频提示词文本，全程合规可过审 | 需保险销售职业场景的故事+配套提示词，且要求场景连贯、合规过审时 |
| 单技能调用 | 输出各独立技能的使用方法，支持用户指定技能编号单独调用对应功能 | 仅需使用某一个核心技能，需查阅该技能单独操作方法时 |
| 组合技能调用 | 展示技能组合调用的逻辑顺序及实操流程，支持用户自定义多技能组合创作需求 | 需按需组合多个技能创作，需查阅组合逻辑、实操流程时 |
| 过审检测 | 单独调用违规侵权检测与过审优化技能，对已有提示词文本做全维度合规检测并输出优化版本 | 已有提示词文本，需做合规性检测、优化为可过审版本时 |
| 提示词模板 | 输出各时长通用短视频提示词模板，标注可替换内容，方便快速适配各类场景 | 需搭建通用提示词框架，希望快速替换内容适配不同场景时 |
| 场景故事创作 | 根据用户指定的主题/场景，创作贴合调性的文字版配套故事，与提示词高度呼应 | 需为短视频提示词创作配套文字故事，强化场景贴合度时 |
| 提示词优化 | 对用户提供的通用提示词，按指定场景/氛围做细节适配与优化，提升画面感与贴合度 | 已有通用提示词，需适配具体场景/氛围、优化细节提升画面感时 |
| 内容整合 | 将创作的故事、提示词、检测结论、核对清单整合为统一格式的文字版文档，方便查阅复用 | 已完成故事、提示词、检测/核对等内容创作，需统一格式整合存档时 |
| 核对校验 | 对已完成的提示词文本做全流程核对，输出核对结论并修正模板适配、衔接流畅性等问题 | 已生成提示词文本，需全流程校验、修正偏差确保可直接使用时 |

## 核心触发规则
1. 无触发关键词时，默认直接输出**合规可过审的视频提示词文本**；
2. 组合调用推荐顺序：模板搭建→故事创作→提示词优化→违规侵权检测与过审优化→核对校验→内容整合；
3. 所有引导流程的选项均为数字/字母，直接输入即可完成需求确认。
