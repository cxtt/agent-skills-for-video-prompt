# 多场景使用案例（快速适配参考）
本技能体系可适配**所有短视频创作场景**，以下为核心场景的**快速使用方法**，基于「关键词触发」和「对话引导」，替换主题/细节即可10分钟完成创作，所有案例均遵循合规可过审原则。

## 一、兴趣爱好类：冲浪（快速生成提示词）
### 方法1：关键词一键触发
直接输入触发关键词：**生成冲浪视频的提示词**
→ 自动生成冲浪主题合规可过审的提示词文本，同步完成过审检测，无需额外操作。

### 方法2：对话引导创作
输入「引导」，依次选择：
3（兴趣爱好类）→ 1（15s）→ 4（9:16）→ 8（热血动感）→ 1（丰富紧凑）→ 2（中密度）→ 5（通用版）→ 2（不需要流量适配）→ 2（仅生成提示词）→ 2（不需要精简标注）
→ 自动生成15s冲浪主题、热血动感氛围、9:16比例的可过审提示词。

## 二、职业场景类：保险销售（5个连贯场景+故事）

→ 自动创作5个保险销售连贯场景的文字版故事+对应提示词文本，全程合规可过审，同步输出检测/核对结论。

## 六、已有提示词：优化+过审检测+比例适配
### 方法：关键词分步触发
1. 输入触发关键词：**提示词优化**，并粘贴已有通用提示词，说明场景/氛围；
→ 自动优化为场景专属提示词，提升画面感；
2. 输入触发关键词：**比例适配+1:1**，并粘贴优化后的提示词；
→ 自动将提示词适配为1:1方形比例，优化镜头构图；
3. 输入触发关键词：**过审检测**，并粘贴比例适配后的提示词；
→ 自动完成全维度合规检测，输出优化后的可过审版本。

## 七、团队创作：内容整合+存档
### 方法：关键词一键触发
完成故事、提示词、检测、核对后，输入触发关键词：**内容整合**，并粘贴所有内容；
→ 自动按固定格式整合为统一的文字版文档，生成唯一标识码，方便团队存档/跨部门对接。

## 八、通用模板：快速适配新场景
### 方法：关键词触发+模板替换
1. 输入触发关键词：**提示词模板**，获取9s/15s/30s/60s/120s通用模板；
2. 按模板中的「{}」替换为新场景的主题/细节/氛围；
3. 输入触发关键词：**过审检测**，对替换后的模板做合规检测；
→ 10分钟完成新场景提示词创作，无需重新构思。

## 九、镜头反向创作：从镜头要素生成故事/提示词
### 方法：关键词触发
输入触发关键词：**镜头反向创作+2-4个镜头+5-12s+9:16+低角度慢推+极慢切+露营+松弛慵懒**
→ 自动根据镜头要素生成贴合的故事和提示词，支持过审检测和比例适配。

## 十、批量处理：多条提示词同时优化
### 方法：关键词触发
输入触发关键词：**批量过审检测**，并粘贴多条提示词（用///分隔）；
→ 自动批量检测多条提示词的合规性，按条目输出优化后的可过审版本。

## 场景适配核心技巧
1. **核心框架不变**：所有场景均复用通用模板的镜头框架、时长分配；
2. **细节专属化**：仅替换场景背景、核心动作、细节描述，贴合目标场景；
3. **氛围匹配**：根据场景调整色调、音乐、拍摄效果，贴合核心氛围；
4. **比例适配**：根据目标平台选择合适的视频比例，优化镜头构图；
5. **合规优先**：所有场景均遵循Skill 6的过审检测规则，避免违规/误判；
6. **流量适配**：过审后可根据平台流量特征做轻优化，提升内容适配性。
