# 多场景使用案例（快速适配参考）
本技能体系可适配**所有短视频创作场景**，以下为核心场景的**快速使用方法**，基于「关键词触发」和「对话引导」，替换主题/细节即可10分钟完成创作，所有案例均遵循合规可过审原则。

## 一、兴趣爱好类：冲浪（快速生成提示词）
### 方法1：关键词一键触发
直接输入触发关键词：**生成冲浪视频的提示词**
→ 自动生成冲浪主题合规可过审的提示词文本，同步完成过审检测，无需额外操作。

### 方法2：对话引导创作
输入「引导」，依次选择：
3（兴趣爱好类）→ 1（15s）→ 8（热血动感）→ 2（仅生成提示词）
→ 自动生成15s冲浪主题、热血动感氛围的可过审提示词。

## 二、职业场景类：保险销售（5个连贯场景+故事）
### 方法：关键词一键触发
直接输入触发关键词：**帮我生成5个连贯场景的保险销售人员的工作**
→ 自动创作5个保险销售连贯场景的文字版故事+对应提示词文本，全程合规可过审，同步输出检测/核对结论。

## 三、职场办公类：工作复盘（30s+暖调治愈）
### 方法：对话引导创作
输入「引导」，依次选择：
2（职场办公类）→ 2（30s）→ 2（暖调治愈）→ 1（需要配套故事）
→ 自动生成30s职场复盘、暖调治愈氛围的提示词+配套故事，合规可过审。

## 四、生活日常类：居家探店（60s+轻松活泼）
### 方法：对话引导创作
输入「引导」，依次选择：
1（生活日常类）→ 3（60s）→ 3（轻松活泼）→ 2（仅生成提示词）
→ 自动生成60s居家探店、轻松活泼氛围的可过审提示词。

## 五、氛围感类：治愈系vlog（15s+松弛慵懒）
### 方法：对话引导创作
输入「引导」，依次选择：
4（氛围感类）→ 1（15s）→ 7（松弛慵懒）→ 1（需要配套故事）
→ 自动生成15s治愈系vlog、松弛慵懒氛围的提示词+配套故事，合规可过审。

## 六、已有提示词：优化+过审检测
### 方法：关键词分步触发
1. 输入触发关键词：**提示词优化**，并粘贴已有通用提示词，说明场景/氛围；
→ 自动优化为场景专属提示词，提升画面感；
2. 输入触发关键词：**过审检测**，并粘贴优化后的提示词；
→ 自动完成全维度合规检测，输出优化后的可过审版本。

## 七、团队创作：内容整合+存档
### 方法：关键词一键触发
完成故事、提示词、检测、核对后，输入触发关键词：**内容整合**，并粘贴所有内容；
→ 自动按固定格式整合为统一的文字版文档，方便团队存档/跨部门对接。

## 八、通用模板：快速适配新场景
### 方法：关键词触发+模板替换
1. 输入触发关键词：**提示词模板**，获取15/30/60s通用模板；
2. 按模板中的「[ ]」替换为新场景的主题/细节/氛围；
3. 输入触发关键词：**过审检测**，对替换后的模板做合规检测；
→ 10分钟完成新场景提示词创作，无需重新构思。

## 场景适配核心技巧
1. **核心框架不变**：所有场景均复用通用模板的镜头框架、时长分配；
2. **细节专属化**：仅替换场景背景、核心动作、细节描述，贴合目标场景；
3. **氛围匹配**：根据场景调整色调、音乐、拍摄效果，贴合核心氛围；
4. **合规优先**：所有场景均遵循Skill 6的过审检测规则，避免违规/误判。
