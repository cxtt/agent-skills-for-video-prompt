# 拓展技能目录（预留）
本目录为**Agent Skills拓展区**，用于存放后续新增的各类创作技能，如：
- 美食/旅行/本地生活等垂直场景短视频提示词技能；
- 直播脚本/图文提示词创作技能；
- 海外平台短视频提示词创作技能；
- 其他AI Agent创作相关技能。

## 目录规范
1. 新增技能需新建**独立子文件夹**，命名格式：「场景/类型-skill」（如food-video-skill、live-script-skill）；
2. 子文件夹内需包含：触发词、技能详情、使用流程、示例等核心文件，参考`core-skills/video-prompt-skill/`结构；
3. 所有技能均需遵循**纯文字输出、高效复用、合规可落地**原则；
4. 新增技能需在本README.md中补充「技能名称+核心能力+更新时间」。
